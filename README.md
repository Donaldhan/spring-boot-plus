# Introduction

## spring-boot-plus

> - 集成Spring Boot 2.1,Mybatis Plus,Druid,FastJson,Redis,Rabbit MQ,Kafka等
> - 可使用代码生成器快速开发项目，使用maven assembly打包发布

- 快速开发
- 灵活配置
- 方便维护
- 快速部署
- 集成缓存
- 集成消息队列
- 集成spring boot
- HTTP请求监控
